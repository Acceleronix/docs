---
title: 'Get Device List'
openapi: 'GET /v2/devicemgr/r3/openapi/product/device/overview'
---

Query the list of devices under a specific product. This endpoint supports pagination and filtering by device status and activation status.

## Authentication Required

This endpoint requires a valid QJWT access token obtained through the authentication API.

**How to use in API Playground**:
1. Copy your complete access token from the authentication response
2. Paste it directly into the "Authorization" field in the API playground
3. The token should start with "QJWT" and include the full JWT string

**Token Format**: `QJWT xxxx`

## Example Usage

Get all devices for a product:

```bash
curl -X GET "https://iot-api.acceleronix.io/v2/devicemgr/r3/openapi/product/device/overview?productKey=YOUR_PRODUCT_KEY&pageNum=1&pageSize=10" \
  -H "Authorization: YOUR_COMPLETE_QJWT_TOKEN" \
  -H "Content-Type: application/json"
```

Filter online devices only:

```bash
curl -X GET "https://iot-api.acceleronix.io/v2/devicemgr/r3/openapi/product/device/overview?productKey=YOUR_PRODUCT_KEY&deviceStatus=1" \
  -H "Authorization: YOUR_COMPLETE_QJWT_TOKEN" \
  -H "Content-Type: application/json"
```
